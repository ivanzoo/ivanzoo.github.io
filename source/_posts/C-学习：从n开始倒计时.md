#### 题目：  
输入一个正整数n,输出从n到0的倒计时  
#### 输入格式：
```
一个正整数n
```
#### 输出格式：
```
输出从n到0之间的所有整数，每个数占一行
```
#### 样例1:
```
输入：
3
```
#### 输出：
```
3
2
1
0
```
#### 解答：
```
    int n;
    cin>>n;
    for(int i=n;i>=0;i-=1)
    {
        cout<<i<<endl;
    }
```
