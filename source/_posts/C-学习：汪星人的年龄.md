#### 题目：  
&emsp;&emsp;汪星人年龄D与人类年龄A的换算公式为：  
A3+D*5  
&emsp;&emsp;也就是说，汪星人1岁相当于人类18岁，汪星人2岁相当于人类23岁。现在已知汪星人的年龄，请写程序计算，它相当于人类多少岁。
#### 输入格式：
```
输入一个整数D，为汪星人的年龄
```
#### 输出格式：
```
输出一个整数A，为换算相当于人类的年龄
```
#### 样例1:
```
输入：
1
```
```
输出：
18
```
#### 解答：
```
   int D;
   int A;
   cin >>D;
   A=13+D*5;
   cout<<A;

```
