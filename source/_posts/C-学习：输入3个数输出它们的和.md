#### 题目：  
输入3个数输出它们的和  
#### 输入格式：
```
输入3个整数
```
#### 输出格式：
```
一个整数（为输入3个整数的和）
```
#### 样例1:
```
输入：
1 2 3
```
```
输出：6
```
#### 解答：
```
    int a,b,c;
    cin >>a>>b>>c;
    cout <<a+b+c;
```
